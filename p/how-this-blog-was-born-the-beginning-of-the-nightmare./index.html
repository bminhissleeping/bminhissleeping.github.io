How this blog was born: the beginning is a nightmare.

Hi guys, this is my first post on this blog. I am going to tell you why this whole blog was made and how I created this blog with mistakes I have made. Ok now lets get straight into the first part:

1. Why this blog was born?

Well, I simply want to find a place to escape from the distraction of the world. The world is filled with many kinds of entertainment like film, games and the most popular kind of entertainment right now is using social media. Don’t get me wrong, social media helps people connect to each other, you can learn new things that you accidentally discover.

However, people say: “The more you get, the less you have” and I am the example of that. I joined every social media platform that I know from facebook, instagram, to X. I casually doomscroll on those platforms just to have news, drama and satisfy myself.

So, what value I have claimed since joining social media? Nothing.

Not only I have gained nothing at that time but also it has made my life worse. My downgraded focus made me feel tired and I couldn’t focus on one task for a long time. My grade was lower than before and I have to have a retake on three subjects: Calculus, Linear algebra and C language coding, the one that I love the most. At that time, I tried to find ways to escape them: deleting plus blocking those apps, doing more work to forget about them. At the end of the day, I still reinstalled those apps and still distracted. I couldn’t let my life go on like this, I know that I need to change.

On 13th Feb, 2026, I decided to make this blog help me have a conversation with you guys so that I can trick my brain that: “Hey, you are using the best social media, you don’t need those distracted platforms anymore” (or at least I think so). I hope this blog can be a place that I can relax after a hard time, find back the old me and get better every day.

That’s all, now let’s come to the next part, which I call “the beginning of the nightmare”. Guess how much time I take to make this blog works as a distracted guy? 5 hours. I know it’d be better if I can focus on only doing that. But in reality, I can’t :).

2. Creating a blog

As a guy choosing information security/cyber security as a main major, I am not going for writing posts and creating my blogs on WordPress or anything like that because of my sensitivity concerning “privacy” on those sites. I choose Git with Hugo to do it. Safe, fast and secure-there’s no reason I can’t refuse. It’s not easy as I think but I am here so I gonna show you how to do it.

Installation

As I said above, I use Git and Hugo so we need to install both.

  1. Hugo Install it like a piece of cake. Use this link and install: Download Hugo It should look like this: Install version depends on your OS. If you are a cool guy, don’t click that link, just do this: winget install Hugo.Hugo.Extended

  2. Git Same as Hugo, click this to install: Download Git It should look like this: For those who still want to be cool: winget install Git.Git

If you use that code and you see the download took so long, you’re not alone. I have the same issue with that. There are many reasons for that problem: Downloaded from different servers, cache/Ram problem, etc.

You still wait and see Git’s still downloading? Just Ctrl C and type winget install Git.Git --interactive to see what’s the problem in order to fix it. If you still can’t fix it? Well then stop acting cool and hit the link to download.

That’s it. I know many people gonna ask: “Where’s the nightmare, are you clickbait or something?”

The post is already long enough (I even feel a bit lazy to read my own post to check mistakes lol) so the nightmare’s on the next post after this one, be ready for that!
